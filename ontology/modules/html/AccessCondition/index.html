Abstract

A controlled vocabulary for specifying access conditions, aligned with COAR Access Rights where applicable.

Table of Contents

  • 1. Introduction
  • 2. Classes
  • 3. Named Individuals
  • 4. Namespaces

1. Introduction

The Access Condition vocabulary defines the access rights associated with resources in the TRIPLE platform, such as Open Access, Restricted Access, or Closed Access. It facilitates the filtering and identification of resource availability.
