# Glossary of Terms - Iteration 14

## External Ontology Alignment for Project Relationships and Citation Networks

| Term | Definition |
|------|------------|
| `frapo:isOutputOf` | A property from FRAPO ontology indicating that a research output is the result of a particular project. |
| `dcterms:isReferencedBy` | A property from Dublin Core Terms indicating that a resource is referenced, cited, or otherwise pointed to by another resource. |
| `triple:Project` | A research project in the Social Sciences and Humanities domain integrated into the GoTriple platform, subclass of schema:Project. |
| `triple:Document` | A document in the GoTriple platform representing scholarly publications and research outputs, with project output relationships. |
| `triple:SemanticArtefact` | A semantic artefact representing formal knowledge representations, extending mod:SemanticArtefact with project and citation relationships. |
| `triple:MediaObject` | A media object such as images, videos, audio, or multimedia content used in SSH research, with project and citation relationships. |
| `triple:Dataset` | A collection of data organized for research purposes in SSH domain, with project output and citation relationships. |
| `owl:allValuesFrom` | OWL restriction specifying that all values of a property must belong to a specific class. |
| `schema:Project` | Schema.org class representing collaborative enterprises planned to achieve particular aims. |
| `schema:CreativeWork` | Schema.org class representing creative works including scholarly publications, datasets, and media. |
| `mod:SemanticArtefact` | Class from MOD ontology representing semantic artifacts such as ontologies and vocabularies. |
